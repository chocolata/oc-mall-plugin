# PDF

Mall bundles the great barryvdh/laravel-dompdf (opens new window) to generate PDF files.

Refer to it's README (opens new window) for usage details.

# PDFMaker Trait

Mall provides a PDFMaker (opens new window) trait that can be used to generate PDF files.

Currently, the Order and Wishlist models implement it by default.

# Configuration

Mall exposes all of the original barryvdh/laravel-dompdf config entries (opens new window) in a separate configuration file.

If you wish to change the default PDF configuration create a config/offline/mall/pdf.php file and return your custom config.

See October's docs on overriding configuration (opens new window).
